# <a name="idebugaliasgetname"></a>IDebugAlias::GetName
このエイリアスの名前を取得します。

## <a name="syntax"></a>構文

```cpp
HRESULT GetName(
   BSTR* pbstrName
);
```

```csharp
int GetName(
   out string pbstrName
);
```

## <a name="parameters"></a>パラメーター
`pbstrName`\
入出力エイリアスの名前。

## <a name="return-value"></a>戻り値
 成功した場合は S_OK を返します。それ以外の場合は、エラーコードを返します。
