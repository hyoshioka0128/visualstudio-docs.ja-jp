# <a name="c26496-use_const_for_variable"></a>C26496 USE_CONST_FOR_VARIABLE
  変数 '% variable% ' が1回だけ割り当てられています。 `const`としてマークしてください。 「 [ C++コアガイドライン con 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)」を参照してください。
