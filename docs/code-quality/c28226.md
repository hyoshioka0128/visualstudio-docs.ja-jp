# <a name="c28226"></a>C28226
警告 C28226: 注釈を PrimOp (現在の宣言内) にすることはできません

 この警告は、名前付き注釈が PrimOp として宣言されており、以前は通常の注釈として宣言されていることを示しています。 注釈に標準のマクロが使用されている場合は、これを行うことはできません。この警告は、通常の使用時には表示されません。
